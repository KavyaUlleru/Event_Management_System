    │

         │
         ├─► Browser asks permission
         │   "Allow location access?"
         │
         ├─► User grants permission
         │
         └─► Route Map Page Loads
             │
             ├─ Gets user GPS coordinates
             ├─ Gets event location
             ├─ Calculates distance (Haversine)
